# Contact Book (Python CLI)

A command-line based Contact Book application built using Python.  
It supports full CRUD operations (Create, Read, Update, Delete) with persistent storage using JSON.

This project focuses on clean logic, data persistence, and proper program flow rather than UI.

---

## Features

- Add new contacts
- Edit existing contacts (name or number)
- Delete contacts with confirmation
- Search contacts by name
- View all saved contacts
- Persistent storage using JSON
- Case-insensitive contact handling

---

## Tech Stack / Concepts Used

- Python
- Dictionaries & data structures
- File handling
- JSON serialization/deserialization
- Functions and control flow
- Menu-driven CLI design
